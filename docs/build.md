build
==============

You can easily build:

linux:

	Test System (x32 & x64): centos >= 5.x, debian >= 5.x
	The compiler gcc: >= 4.4.x


	[root @ localhost ~] # cd $KBE_ROOT/kbe/src

	[root @ localhost/ src] #make
